Plugin para WooCommerce que preenche automaticamente o endereço com base no CEP e permite gerenciar até três endereços personalizados por cliente: residencial, comercial e presente.

== Description ==

Este plugin melhora a experiência do usuário no WooCommerce ao permitir:

* Preenchimento automático do endereço a partir do CEP.
* Validação de bairros com base em faixas de CEPs personalizadas.
* Mensagens personalizadas sobre cobertura de entrega e frete grátis.
* Gerenciamento de até 3 endereços por cliente: residencial, comercial e presente.
* Substituição da opção padrão de "entregar em outro endereço" por uma seleção personalizada.
* Ideal para negócios locais que atendem regiões específicas, como bairros da cidade de São Paulo.

== Installation ==

1. Envie a pasta `meu-endereco-personalizado` para o diretório `/wp-content/plugins/`.
2. Ative o plugin no menu "Plugins" do WordPress.
3. Vá até o painel administrativo em **Endereço Personalizado**.
4. Cadastre os bairros e faixas de CEP que deseja atender.
5. Pronto! O plugin está ativo no carrinho e checkout do WooCommerce.

== Frequently Asked Questions ==

= Funciona fora da cidade de São Paulo? =

Sim. Você pode cadastrar os bairros e faixas de CEP de qualquer região.

= Este plugin substitui os campos padrão de endereço de entrega? =

Sim. Ele oferece uma forma personalizada de gerenciar endereços no checkout, eliminando a opção padrão de “entregar em outro endereço”.

= O plugin depende do WooCommerce? =

Sim. Ele é desenvolvido especificamente para funcionar com o WooCommerce.
